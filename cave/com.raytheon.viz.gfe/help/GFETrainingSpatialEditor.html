Spatial Editor

The Spatial Editor displays one or more grids in plan-view format.  This editor allows you to modify individual grid points using a number of tools, built-in to the GFE and tools that you write yourself.  Before you start any exercises using the Spatial Editor, it is very useful to know how to control the Spatial Editor display .
